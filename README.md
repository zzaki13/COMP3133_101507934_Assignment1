## Project Description

This project is an Employee Management System backend built using Node.js, Express, GraphQL, and MongoDB Atlas.

It allows users to register, login, and perform CRUD operations on employees.

## Features

User:
- Signup
- Login

Employee:
- Add Employee
- View Employees
- Search Employee by ID
- Search Employee by Designation or Department
- Update Employee
- Delete Employee
